
# 자바 스윙 쿠키런




## 기능

- 체력게이지
- 점수
- 점프
- 더블점프
- 슬라이드
- 낙하
- 일시정지

젤리를 먹으면 이펙트가 생기고 점점 사라진다

일정 거리를 달리면 체력이 조금씩 줄어든다

물약을 먹으면 체력이 일정량 늘어난다

발판, 공중발판 - 발판이 있으면 떨어지지 않는다

배경화면이 천천히 흐른다

맵이 바뀌면 배경화면이 페이드아웃 페이드인 된다

장애물에 부딛히면 캐릭터가 반투명해지며 무적상태가 된다

esc를 누르면 일시정지되고 esc를 다시 누르면 게임이 진행된다
